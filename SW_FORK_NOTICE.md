# SWLauncher

SWLauncher é uma versão modificada não oficial baseada no projeto
Zalith Launcher 2.

## Aviso

Este projeto:

- não é afiliado oficialmente ao Zalith Launcher 2;
- não é afiliado ao PojavLauncher;
- não é afiliado à Mojang Studios ou Microsoft;
- preserva a licença GPL-3.0 e os avisos de copyright aplicáveis;
- utiliza um nome e identificador Android próprios.

## Base

- Projeto original: Zalith Launcher 2
- Motor de lançamento derivado do ecossistema PojavLauncher
- Licença: GPL-3.0

## Identidade atual

- Nome: SWLauncher
- Versão: 0.0.2-r1
- Application ID Debug: com.swteam.swlauncher.debug
- Application ID Release futuro: com.swteam.swlauncher
- Arquitetura de desenvolvimento inicial: armeabi-v7a
